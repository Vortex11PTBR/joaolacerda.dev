Requisitos Recomendados

Especificações mínimas para instalação dos nossos produtos.

🪟 Windows

  • Sistema: Windows 7 ou superior
  • Processador: Intel/AMD 1.8GHz+
  • RAM: Mínimo 2GB (recomendado 4GB)
  • Disco: 500MB disponível
  • Resolução: 1024x768 mínimo

🐧 Linux

  • Distribuição: Ubuntu 18.04+
  • Processador: Intel/AMD 1.8GHz+
  • RAM: Mínimo 2GB (recomendado 4GB)
  • Disco: 500MB disponível
  • Resolução: 1024x768 mínimo

🍎 macOS

  • Sistema: macOS 10.12 ou superior
  • Processador: Intel 1.8GHz+
  • RAM: Mínimo 2GB (recomendado 4GB)
  • Disco: 500MB disponível
  • Resolução: 1024x768 mínimo

ℹ️ Informações Importantes

  • Para melhor desempenho, recomendamos ter 4GB de RAM ou mais
  • Certifique-se de ter conexão de internet estável para atualizações
  • Dependências adicionais podem ser necessárias em alguns sistemas
  • Para suporte técnico, entre em contato conosco
