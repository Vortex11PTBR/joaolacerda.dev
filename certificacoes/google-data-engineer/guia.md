# Google Professional Data Engineer
**Custo: R$0,00 via Get Certified (outubro/2025) | Validade: 2 anos | Idioma: Inglês**

---

## ✅ Como conseguir GRÁTIS via Get Certified

### Passo 1 — Crie conta no Google Cloud
1. Acesse https://cloud.google.com
2. Crie sua conta Google Cloud (gratuita, $300 de créditos de boas-vindas)

### Passo 2 — Inscreva-se no programa Get Certified
- Acesse https://cloud.google.com/innovators/getcertified
- Próxima edição: **outubro a dezembro de 2025**
- Fique de olho para se inscrever assim que abrir (vagas limitadas)
- Ative notificações na página ou siga @GoogleCloud no LinkedIn

### Passo 3 — Complete a trilha de estudos do programa
- O programa fornece trilha guiada com labs, cursos e simulados
- Complete tudo dentro do prazo da edição
- O voucher é entregue ao concluir a trilha

### Passo 4 — Agende e faça a prova
1. Acesse https://cloud.google.com/certification/data-engineer
2. Use o voucher para agendar sem custo
3. Prova online supervisionada ou presencial
4. Duração: 2 horas | 40-50 questões múltipla escolha

## 📚 O que estudar

### Domínios do exame:
| Domínio | Peso |
|---|---|
| Projetar sistemas de processamento de dados | ~22% |
| Ingerir e processar dados | ~25% |
| Armazenar dados | ~20% |
| Preparar e usar dados para análise | ~15% |
| Manter e automatizar workloads | ~18% |

### Serviços principais para estudar:
- **Ingestão:** Pub/Sub, Dataflow, Cloud Data Fusion
- **Storage:** BigQuery, Cloud Storage, Bigtable, Firestore, Cloud SQL, Spanner
- **Processamento:** Dataflow (Apache Beam), Dataproc (Hadoop/Spark), Dataform
- **ML:** Vertex AI, BigQuery ML, AutoML
- **Orquestração:** Cloud Composer (Apache Airflow)
- **Segurança:** IAM, VPC, Cloud KMS, Data Loss Prevention (DLP)
- **Monitoramento:** Cloud Logging, Cloud Monitoring
